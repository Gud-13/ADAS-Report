ADAS Shift Report
Quick Start Guide
📋 Shift Report
1
Fill in Shift Parameters
Date, Vehicle ID, Mission ID, City, Country, Driver & Operator ID
2
Add events with Quick Add
Tap any button — event is added with current time. Yellow = System, Blue = Organisational
3
Fill in each event card
Time start/end · KM (Enter copies start→finish) · Address (📍 for GPS) · For DT events: pick comment from red dropdown, edit with ▾
4
Tap Preview & Copy
Full shift report is copied to clipboard — paste anywhere
💾 Disk Status
• Tap + Add disk for each SSD drive
• Set status: empty · in use · full
• For in use: enter available TB → % calculates automatically from 90 TB total
• Tap 📋 Copy report to send disk status via WhatsApp
• Disk data is preserved when starting a New Shift
⟳ New Shift
Clears all events and resets parameters. Disk status stays. Use this at the start of each new shift day.
🎫 Ticket
1
Set up Ticket Settings once
Team Leader, Vehicle, License Plate, Phones, Location — saved permanently
2
Fill per-ticket fields
Category · short Description · Body text (the "Dear Partners" message)
3
Tap 📋 Copy Ticket
Ready to paste in WhatsApp or email
💡 Tips
• Page refreshes safely — all data is auto-saved
• 📍 GPS button fills the address automatically (allow location in browser settings)
• Enter in any field confirms and closes the keyboard
• KM Start → Enter copies the value to KM Finish automatically
